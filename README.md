# joguito

## Starting Development Server

Using python:
```
python -m http.server 9000
```

Using PHP ([How to Install PHP](https://www.php.net/manual/en/install.php)):
```
PHP -S localhost:9000
```
<br>
and open the game using your browser and typing:

```
localhost:9000
```
